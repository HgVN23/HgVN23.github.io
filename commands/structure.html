/structure
Công dụng
Thay thế cho Structure Block, để Save hoặc Load công trình
Đối tượng có thể dùng Command
Player có OP
Chủ Map
Addon
/execute
/function
/gametest
Command Block
Cú pháp
/structure delete <name: string>
delete là xóa
<name: string> là id cấu trúc
/structure delete wiki:hg — xóa công trình wiki:hg
/structure load <name: string> <to: x y z> [rotation: Rotation] [mirror: Mirror] [animationMode: StructureAnimationMode] [animationSeconds: float] [includesEntites: Boolean] [includesBlocks: Boolean] [integrity: float] [seed: string]
load là hiện
<name: string> là id cấu trúc
<to: x y z> là tọa độ hiện
[rotation: Rotation] là xoay. Danh sách:
0_degrees — 0°
90_degrees — 90°
180_degrees — 180°
270_degrees — 270°
[mirror: Mirror] là đối xứng. Danh sách:
none — không
x — trục tọa độ x
xz — trục tọa độ xz
z — trục tọa độ z
[animationMode: StructureAnimationMode] là hoạt ảnh cấu trúc xuất hiện. Danh sách:
block_by_block — từng Block
layer_by_layer — từng tằng
[animationSeconds: float] là khoảng thời gian Delay của hoạt ảnh cấu trúc xuất hiện
[includesEntites: Boolean] là chứa Entity. Danh sách:
false — tắt
true — bật
[includesBlocks: Boolean] là chứa Block. Danh sách:
false — tắt
true — bật
[integrity: float] là tỉ lệ load được hết công trình
[seed: string] cần 1 ai đó giải thích hộ
/structure load wiki:hg 1 1 1 0_degrees none block_by_block 5 false true 100 — hiện công trình wiki:hg từ tọa độ 1 1 1, không xoay, không đối xứng, hiện từng Block delay 5 giây, không có Entity, có Block và 100% Load hết

/structure save <name: string> <from: x y z> <to: x y z> [includesEntites: Boolean] [saveMode: StructureSaveMode] [includesBlocks: Boolean]
save là lưu
<name: string> là id cấu trúc
<from: x y z> là tọa độ bắt đầu
<to: x y z> là tọa độ kết thúc
[includesEntites: Boolean] là chứa Entity. Danh sách:
false — tắt
true — bật
[saveMode: StructureSaveMode] là cách lưu. Danh sách:
disk — lưu vĩnh viễn, Rejoin Map không mất
memory — lưu tạm thời, Rejoin Map sẽ mất
[includesBlocks: Boolean] là chứa Block. Danh sách:
false — tắt
true — bật
/structure save wiki:hg 0 0 0 2 2 2 false disk true — lưu công trình trong id wiki:hg từ 0 0 0 đến 2 2 2, không có Entity, lưu trong Disk và có Block
/structure save <name: string> <from: x y z> <to: x y z> [saveMode: StructureSaveMode]

<to: x y z> là tọa độ kết thúc
