/ability
Công dụng
Chỉnh khả năng của Player như bay hoặc đặt/phá Block hoặc Chat
Đối tượng có thể dùng Command
Player có OP
Chủ Map
Addon
/execute
/function
/gametest
Command Block
Cú pháp
/ability <player: target> <ability: Ability> <value: Boolean>
<player: target> là mục tiêu (Player)
<ability: Ability> là khả năng. Danh sách:
mayfly là khả năng bay
mute là khả năng mất Chat
worldbuilder là khả năng đặt/phá Block
<value: Boolean> là phần trạng trái. Danh sách:
true là bật
false là tắt
/ability @a mayfly true — All Player sẽ có khả năng bay
/ability <player: target> [ability: Ability]
<player: target> là mục tiêu (Player)
[ability: Ability] là khả năng. Danh sách:
worldbuilder là khả năng đặt/phá Block
mayfly là khả năng bay
mute là khả năng mất Chat
/ability @a mayfly — All Player sẽ có khả năng bay. Cấu trúc này sẽ bớt phần true nhưng để tắt khả năng đó đi thì cần thêm false cuối Command
Lưu ý
Cần bật chế độ EDU để dùng Command
worldbuilder đang bị lỗi nên không thực hiện được bật/tắt
Thoát Map và vào lại thì phải dùng lại lệnh đế bật/tắt khả năng lại
Khi chuyển gamemode sẽ reset lại khả năng
